# Exercise 9

Show the output produced by each of the following program fragments. Assume that `i`, `j`, `k` are `int` variables.

(a)

```c
i = 7; j = 8;
i *= j + 1;
printf("%d %d", i, j);
```

(b)

```c
i = j = k = 1;
i += j += k;
printf("%d %d %d", i, j, k);
```

(c)

```c
i = 1; j = 2; k = 3;
i -= j -= k;
printf("%d %d %d", i, j, k);
```

(d)

```c
i = 2; j = 1; k = 0;
i *= j *= k;
printf("%d %d %d", i, j, k);
```
